# Covid19 Dashboard Italy

A Vue.js Dashboard of Covid-19 in Italy
Built with Vue.js - Vuetify - ApexCharts

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
